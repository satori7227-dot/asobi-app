# ASOBI 2026 Q4 マーケティングロードマップ v1（2026-06-24 起案）

v1.0 リリース後の季節キャンペーン計画。Heads Up! / Jackbox / Psych! が 10 年生存している共通要素「季節 connected コンテンツ追加 + 透明価格」を ASOBI に適用。Bunch / Marco Polo / Picolo の死因「後付けペイウォール化」を構造的に回避する宣言を含む。

## 信頼貯金宣言（リリース時 App Store 説明文・LP 冒頭に明記）

> **ASOBI は買い切り後の機能を有料化しません**。
> v1.0 で無料配布したシーン / お題 / ツールは、後から課金壁の向こうに移すことを永続的に禁止します。
> 新しいシーン / お題パックは追加デッキとして任意購入してください（v1.1 以降）。

これにより Bunch / Marco Polo / Picolo の「後付けペイウォール化」失敗の信頼貯金を逆吸収する。
