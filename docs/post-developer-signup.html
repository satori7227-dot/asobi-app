Apple Developer 契約直後にやること

契約日: ______（記入）／ 想定所要: 初日 2-3 時間 + その後 1 週間。
これは next-actions-dashboard の「配布・公開」角度を「契約済」状態の前提で具体化した手順書。

0. 即日（契約完了の通知が来たら）

MUST15分2要素認証と本人情報の確認

  • appleid.apple.com で 2FA が信頼デバイスベースで有効か確認
  • 姓名が戸籍漢字 or ローマ字に統一されているか（ニックネーム/屋号は審査落ち）

MUST20分App Store Connect Agreements / Tax / Banking

  1. App Store Connect → Business → Agreements
  2. Free Apps Agreement 自動同意確認、Paid Apps Agreement に署名
  3. Tax タブで Japan Tax Forms 提出（未提出だと売上 20% 強制源泉徴収）
  4. Banking タブで 本人名義 銀行口座を登録（屋号口座・法人口座 NG）

MUST5分Small Business Program 申請

  • App Store Connect → Agreements → "Apply" Small Business Program
  • 新規開発者は自動適格、翌月から手数料 30% → 15% に

1. 初週（コード公開準備）

MUST10分Bundle ID 確定とアプリレコード作成

  • Certificates → Identifiers → App IDs で com.idogawa.Asobi を Explicit 登録
  • App Store Connect → My Apps → + → 新規アプリ作成
  • Primary Language: Japanese (Japan)、SKU は asobi-001 など決定（後変更不可）

MUST30分fastlane の API Key を設定

  1. App Store Connect → Users and Access → Keys → + で App Manager 権限の API Key 発行
  2. .p8 ファイルをダウンロード（再ダウンロード不可、1Password 等に即保管）
  3. 環境変数を ~/.config/asobi/.envrc 等に設定（git 管理外）: 
    export FASTLANE_USER=satori7227@gmail.com
export ASC_KEY_ID=XXXXXXXXXX
export ASC_ISSUER_ID=YYYYYYYY-...
export ASC_KEY_FILEPATH=$HOME/secure/AuthKey_XXXXXXXXXX.p8
  4. 確認: fastlane test が通れば API 接続 OK

MUST30分project.yml に DEVELOPMENT_TEAM を入れる

  • Apple Developer → Membership ページから Team ID をコピー
  • project.yml の settings.configs.Release.DEVELOPMENT_TEAM に貼る
  • xcodegen generate で反映

MUST1時間初回 TestFlight 配信

  1. fastlane bump（ビルド番号自動採番）
  2. fastlane beta（Archive → TestFlight 自動アップロード）
  3. App Store Connect → TestFlight → Internal Testing にビルド反映確認（5-10分）
  4. iPhone の TestFlight アプリで実機検証

2. 申請準備（2 週目）

MUST2時間App Privacy ラベル登録

  • App Store Connect → App Privacy → Get Started
  • Data Not Collected を選択（ASOBI は実装上これが正解）
  • Privacy Manifest (Asobi/PrivacyInfo.xcprivacy) と申告が一致していることを確認

MUST30分Age Rating + メタデータ

  • Age Rating アンケート: 「Infrequent/Mild Alcohol」「Infrequent/Mild Mature」を選んで 12+ に
  • App Subtitle / Promotional Text / Description / Keywords を入力（launch-prep ガイド参照）
  • Privacy Policy URL: GitHub Pages の URL
  • Support URL: 意見箱用ページ or Notion 公開ページ

MUST1時間スクリーンショット撮影

  • iPhone 16 Pro Max (6.9インチ) + iPhone Plus サイズ (6.5インチ) 各 3-10 枚
  • 1枚目「シーンで選ぶ・1191ゲーム・オフライン」、2枚目「全シーン横断」、3枚目以降は具体例
  • Simulator から Cmd+S で取得 → サイズ違反になりやすいので Simulator サイズに従う

3. IAP 解禁（任意・利益が出始めたら）

SHOULD2時間サブスク 3 ティア + Tip Jar 3 ティア登録

  • App Store Connect → 自分のアプリ → In-App Purchases & Subscriptions
  • サブスク: asobi.sub.month ¥120/月 / asobi.sub.year ¥1,100/年
  • 買切: asobi.allin ¥1,800
  • Tip Jar 消費型: com.idogawa.Asobi.tip.small / medium / large ¥120/¥600/¥1,200
  • 登録後、Constants.swift の premiumEnabled / tipJarEnabled / softPaywallEnabled を順次 true に

SHOULD30分Family Sharing と 7日無料トライアル設定

  • 各サブスク/買切で Family Sharing を ON
  • 年プランに Introductory Offer「Free Trial 7 days」を設定

4. リリース実行（準備整ったら）

MUST10分Submit for Review

  • Phased Release for Automatic Updates を ON（1→2→5→10→20→50→100% を 7 日かけて）
  • 初回は Manual Release 推奨（審査通過後に自分のタイミングで公開）

MUST24hResolution Center 監視

  • リジェクト通知が来たら 24 時間以内に応答（テンプレ参照）
  • 3 営業日以内に Apple から最終判定が来るのが通常
